第2章 线性表

November 14, 2022 · 2 min · 255 words · Chan | Suggest Changes

  1. 线性表是逻辑结构，顺序表链表是存储结构

  2. 给定n个元素的一维数组，建立一个有序单链表的时间复杂度最小是 O(nlog~2~n)：排序最短O(nlog~2~n)，建表O(n)

  3. 时间复杂度分析尾插头删
    单链表/双链表O(n)O(1)
    不带头结点的单循环链表O(n)O(n)
    不带头结点有尾指针的单循环链表O(1)O(1)

  4. 有无头结点对头插的影响：没有头结点，需要遍历到表尾才能将尾节点指向插入的结点，复杂度为O(n)；有头结点为O(1)

概念#

逻辑结构（定义）：具有相同数据类型的n（==>=0==）个数据元素的有限序列

物理结构：

  • 顺序存储

    • 顺序表
      • 静态分配空间
      • 动态分配空间

  • 链式存储

    • 单链表
    • 双链表
    • 循环链表
    • 静态链表（借助数组实现）

基本操作：

- InitList(&L)
- Length(L)
- LocateElem(L, e)
- GetElem(L, i)
- ListInsert(&L, i, e)
- ListDelete(&L, i, &e) 用e返回删除的元素
- PrintList(L)
- Empty(L)
- DestroyList(&L)

顺序表#

特点：元素的逻辑顺序和物理顺序相同，随机存取，存储密度高，插入删除需移动大量元素。

实现方式#

静态分配#

typedef int ElemType;
#define MaxSize 50
typedef struct {
	ElemType data[MaxSize];
	int length;
}Sqlist;

动态分配#

#define MaxSize 50
#define InitSize 50
typedef struct {
	ElemType *data;
	int length;
}Sqlist;

Sqlist L;
L.data = (ElemType*)malloc(sizeof(ElemType)*InitSize);
L.data = new ElemType[InitSize];

	 if(A.length + B.length > C.maxSize)
	 	return false;
	 int i=0, j=0, k=0;
	 while(i<A.length && j<B.length) {
	 	if (A.data[i] < B.data[j])
	 		C.data[k++] = A.data[i++];
	 	else
	 		C.data[k++] = B.data[j++];
	 }
	 while (i < A.length)
	 	C.data[k++] = A.data[i++];
	 while (j < B.length)

	 C.length = k;
	 return true;
}

逆置#

  • 原地逆置O(1)（不使用栈）

    • 解法一：头插法

      LinkList Reverse_HeadInsert(LinkList L) {
	Lnode *p = L->next, *r;

	L->next = NULL;
	while(p!=NULL){
		r = p->next;

		p->next = L->next; // 头插
		L->next = p;

		p = r;
	}
	return L;
}

    • 解法二：反转指针

      LinkList Reverse_RevertPointer(LinkList L) {
	LNode *pre = NULL, *p = L->next, *r = L->next;
    while (p!=NULL) {
        r = r->next;
        p->next = pre;
        pre = p;
        p = r;
    }
    L->next = pre;
    return L;
}
