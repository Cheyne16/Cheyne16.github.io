3 记号的描述与识别#

词法与正规文法#

  • 词法：描述语言的标识符、常数、运算符、标点符号等记号的文法 —— 正规文法
  • 语法：借助记号来描述语言的结构的文法 —— 上下文无关文法

记号的文法#

写出正规表达式/定义式，再由此写出正规文法

  • 标识符：一般写作： id -> letter rid ; rid -> 3 / letter rid / digit rid [rid: (letter/digit)*]
  • 常数
    • 整数：digits -> digit remainder ; remainder -> 3 / digit remainder [remainder: (digit)*]
    • 无符号数：很复杂
  • 运算符：只讲了关系运算符
  • 分界符
  • 关键字

词法分析程序的构造#

每个状态对应一段程序。

在当前状态根据输入转到下一状态，执行对应的程序。

  • 在状态0，若读到空格（blank、tab、enter）则继续，直到读到一个非空字符为止；接下来就是上面的过程
  • 在标识符状态，识别并组合出一个标识符后，还要加入一些动作。

编制词法分析程序#

根据状态转换图，设计自动机，实现状态的转换及各个状态的处理。

  • 初始状态，读入空格后继续直到非空字符 —- 解决了忽略空格这个任务
  • 在各种记号的识别流程中，当读到下一部分的第一个字符时，结束读取，回退向前指针，状态设为初始状态，分析处理刚才识别出的信号
  • 注意两个特殊状态的处理细节：初始0状态 和 标识符状态
